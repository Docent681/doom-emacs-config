# Doom Emacs Config

This is a basic config of a 100% noobie for Doom Emacs.
I started using emacs only at spring 2026, don't expect anything serious from this config.
If you're not familiar with Doom Emacs,  [check repo here](https://github.com/doomemacs/doomemacs).

## Description

At the moment, I've implemented only the bare essentials for working on c/c++ and python projects:
- enabled most useful packages in init.el like vterm, lsp, treemacs etc;
- added package for integration of Cyrillic alphabet in evil mode;
- added lsp-servers for autocorrection and code completion;
- configured usage of default icons and fonts for doom.

## Usage

I created this repo so I wouldn't loose my config. However, if you consider downloading it for some reason, after updating your emacs config, don't forget to run:

```
doom sync
```

and restart doom emacs.
